# Projeto de classificação de textos baseado em LLM

A classificação foi realizada com base no modelo facebook/bart-large-mnli, escolhido por sua robustez no tratamento de problemas de classificação de textos zero-shot.

O modelo facebook/bart-large-mnli se trata de um modelo de redução de ruído para geração, tradução e, mais importante para este caso, compreensão de linguagem natural.

A escolha desse modelo, ainda, passou pelo critério de custos, utilização e entendimento do dia a dia. Sabendo da utilização recorrente de LLMs, a opção pelo uso de um LLM se deu para promover a aplicação dessa tecnologia e avaliação de formas de treinamento. A opção por um modelo gratuito foi pensada para reduzir o investimento feito na aplicação do modelo especialmente devido à característica introdutória do problema.
